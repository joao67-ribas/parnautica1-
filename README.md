
# Parnautica-Catálogo de barcos,quadriciclos e motores 

Descrição

Site desenvolvido para a Parnautica com o objetivo de apresentar barcos, quadriciclos e motores de forma organizada e profissional, permitindo que clientes visualizem os produtos disponíveis e entrem em contato para obter mais informações.


##  Funcionalidades
* Catálogo de barcos
* Catálogo de quadriciclos
* Catálogo de motores
* Galeria de imagens
* Informações de contato
* Layout responsivo para celular e computador

## Tecnologias utilizadas
* HTML5
* CSS3
* JavaScript





## Objetivo do projeto 
Criar uma vitrine digital moderna para a Parnautica, facilitando a apresentação dos produtos aos clientes.
